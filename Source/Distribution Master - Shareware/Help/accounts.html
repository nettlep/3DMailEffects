The accounts menu allows you to select which accounts you want to view email for.

    • -- All accounts -- Select this to view email for all of your accounts. If you have many accounts and want to limit the application window to only display the emails for a single account, then select one of the account names from the list below.

      Note that when a single account is chosen, mail is still checked for other accounts, but is not displayed. Also note that the Check for mail now option (in the context menu) will only force a check for the selected account.
