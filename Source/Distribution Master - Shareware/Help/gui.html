Before we begin, it's important that you understand that 3DME is a skinned application. A "skinned" application is one that can have its interface modified, changed, rearranged, and in general, be made utterly unrecognizable. Because of this, it's impossible to document all the possible skins that one might find on the internet. Instead, I'm going to ask that you use the Default skin when reading this documentation. The default skin looks like this:

Spend a moment and run your mouse cursor around the window slowly. Many of the graphical elements are clickable items. For example, if you place the cursor over the word "Exit" (near the top-right corner of the window) you'll notice that it changes into a link cursor (much like hovering over a link on a web page.)

Each time you load a new skin, it's important to do this, in order to get aquainted with how that skin is arranged. Skins can be created by anybody and they're fun. To find out more, click here. Note that each skin has the ability to completely rearrange the interface. This means that somebody could create a skin that doesn't contain an exit button. As a matter of fact, a skin can be created that doesn't even contain any buttons, or even an email list. A user may create a skin that contains nothing more than the status line, if they wish. However, the complete functionality of 3DME is not lost (though it is somewhat impared) because you can always right-click on the tray icon to get the context menu. You can also right-click on the background of the skin. The background is any part of the interface that is:

  • Not occupied by a button (i.e. the cursor is not a hand-cursor)
  • Not occupied by the email list.

Each of the buttons on the skin mirror the functionality of a function within the context menu. For example, the above skin contains a series of 4 round iconic buttons. Here are their associated functions (listed from left-to-right):

Display the context menu
Check for new mail (without waiting)
Mark the visible messages as read
Disable All Checking

Of particular interest, should be the email list. From this list, you can delete emails, preview emails and more. Simply hilight a series of emails and right-click in the selection area. If no emails are selected, simply right-click on a single email to perform operations specific to that email.
