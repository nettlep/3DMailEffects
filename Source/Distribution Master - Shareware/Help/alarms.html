The purpose of the Alarm preferences is to allow you to define what will happen when an alarm is triggered. Alarms are triggered when new mail arrives on an account. You can choose to do nothing, or you can cause 3DME to notify you in some fashion.

      Popup window with alarm -- Causes 3DME to come to the foreground, and if hidden, unhide itself.

      Popup 3D effect with alarm -- Enable this option if you want to see the 3D object for alarm notification.

      Sound an alarm when new mail arrives -- Put simply, play a sound with the alarm.

      WAV file -- If Sound an alarm when new mail arrives is selected, you may enter a sound file here. Click on the "..." button to browse for a WAV file, and click on the speaker icon to test the sound that you've chosen.

      Run a program with this alarm -- This allows you to setup Runs to execute with new mail. Runs are nothing more than a small batch of programs (or just one program) to execute.
