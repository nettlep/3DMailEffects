The toggles menu contains a series of toggles (things you turn on/off, or disable/enable.) Note that some of these features are found elsewhere, but duplicated here for easy access. In those cases, this documentation will direct you to the proper place for a full explanation of their functionality.

    • Snap to desktop edges -- Visit the preferences dialog for information on this option.

    • Show new mails only -- With this option enabled, you will only see new mails in the main application window. Emails that have been marked as read will not appear in the application window. Note that if this option is enabled, and you mark read (see above), that the email list in the main application window will be empty, even though there are emails on your mail server.

    • Stay on top -- Visit the preferences dialog for information on this option.

    • Disable keyboard LED blinking -- Visit the preferences dialog for information on this option.

    • Disable icon tray blinking -- Visit the preferences dialog for information on this option.

    • Disable all application popups -- The alarms dialog allows you to enable 3DME to pop up (appear, if hidden) when new mail arrives. If you wish to temporarily disable this behavior, simply select this option.

    • Disable all 3D popups -- The alarms dialog allows you to enable 3DME to display (popup) a 3D object when new mail arrives. If you wish to temporarily disable this behavior, simply select this option.

    • Disable all alarms -- Select this option to disable all alarm activity. This includes popups, sounds, programs set to run with an alarm, etc.

    • Disable all alarm programs -- Similar to Disable all alarms, except that this only disables programs configured to run when the alarm is triggered.

    • Disable all reminders -- Select this option to disable all reminder activity. This includes sounds, programs set to run with a reminder, etc.

    • Disable all reminder programs -- Similar to Disable all reminders, except that this only disables programs configured to run when the reminder is triggered.

    • Disable checking for email -- Select this option to temporarily discontinue all email checking. You may wish to do this when you disconnect (if you're using a modem) or if you wish to completely silence 3DME for a period of time. I tend to use this when I'm on the phone. Note that when this option is selected, the status line on the application window and the tool-tip for the tray icon will both contian the string "[DISABLED]" as a reminder that your mail is not being checked.
