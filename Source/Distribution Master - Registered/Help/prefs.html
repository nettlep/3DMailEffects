This dialog can be accessed from the context menu or when you start 3DME without any accounts setup (this will happen every time in the shareware version.)

The preferences dialog has two main purposes. First, it allows you to add and maintain your account information within 3DME. But it also allows you to setup the way 3DME works on a global scale (i.e. settings that affect all accounts.) We'll start with the Email Account Management...

    • Add -- add an account to the list. Clicking on this button will take you to the Account preferences dialog.

    • Remove -- remove a selected account from the list. This button will be disabled until an account is selected.

    • Edit -- edit the selected account. Clicking on this button will take you to the Account preferences dialog (just as Add does, but this time, the account information will be filled in for you.) You may also double- click on an entry in the list to edit it.

      The following options are either enabled ("checked") or disabled (not-checked). Here is what each of the entries means:

    • Automatically run on startup -- Causes 3DME to insert a reference to itself in registry, which causes it to automatically start when you start Windows.

    • Confirm Deletes -- When you select an email (or group of emails) to be deleted, 3DME will first ask you if you're sure you want to.

    • Show tips on Startup -- When 3DME starts up, it will automatically show you a Tip of the day. You can also access this from the context menu.

    • Stay on top -- Causes 3DME to remain the top-most window. It will never be hidden behind any other window. You can also access this from the context menu.

    • Mark all read when hiding -- If this option is enabled, any emails in the list that are new will automatically be marked as read. This means that the keyboard LEDs (if enabled) will no longer flash, the tray icon will stop flashing, and reminders will no longer sound.

    • Close 3D popup on hide -- If this option is enabled, the 3D object will be be closed (if it's open) when you hide 3DME.

    • Auto-check -- Enable this if you want 3DME to notify you when it checks for updates automatically. This is done by querying the FluidStudios web server for a single TXT file. During the automatic update check, you will see a dialog that explains that 3DME is checking for an update. If no update is found, that dialog will automatically go away. If an update is found, it will open a new dialog that shows the revision history for 3DME (what has changed in the new update.) This dialog will remain until you close it. The update will happen each time you start 3DME, and then once every 7 days from that point forward.

    • Start minimized -- If this option is enabled, 3DME will automatically minimize itself when it starts.

    • Log debugging information to TRANSACTION.LOG file -- Normally, you will leave this option disabled. However, when troubleshooting problems (or reporting problems to 3DME Support), this information can be a great aid. When this option is enabled, all networking transactions (sends & receives) are logged to a file called TRANSACTION.LOG that is located in the 3DME installation directory. If you're troubleshooting a rare problem, it may be advantageous to leave this option enabled until the problem occurs. You can do so without fear of wasting disk space, because the file is truncated (i.e. reset to 0 length) if it reaches a size beyond 4MB. Also note that critical errors will be logged to this file even if this option is disabled.

    • Blink the keyboard LEDs -- If this option is enabled, the keyboard LEDs will blink when there are new mails pending. Note that the keyboard LEDs will not flashing if 3DME is not the active window, because this would impede your ability to use the keyboard while working in another window. The slider allows you to control the rate of the keyboard blinking. You can also disable this item from the context menu.

    • Blink the try icon -- If this option is enabled, 3DME will blink the try icon (see the image below). The slider allows you to control the rate of the tray icon blinking. You can also disable this item from the context menu.

    • Snap to desktop edges -- Enable this option if you want 3DME to snap to the desktop edges. For example, as you move the 3DME window around the screen, it will snap to the sides & top of the desktop when it gets close to one of the borders. Another way to think about this, is as if 3DME was magnetized to the border of the screen. This feature allows you to place 3DME directly on the edges or in the corner of the desktop with precision. The slider allows you to adjust the sensitivity of the magnet. You can also enable this item from the context menu.
Note that these settings are saved, but only when 3DME exits normally. If the system crashes, or if you turn the computer off without properly shutting it down (always a bad idea), or if you close 3DME from the task manager, then 3DME will not save these settings.
