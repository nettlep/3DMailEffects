The context menu exposes the majority of the functionality for 3DME.

    • 3D Mail Effects [version] -- Tells you the version of 3DME that you're running. Clicking on this will do nothing (it won't even close the context menu.)

    • Toggles -- Opens up the toggles menu which gives you immediate access to some of 3D Mail Effects' configuration settings (also found in the preferences dialog) as well as access to a few other settings.

    • Accounts -- Opens up the accounts menu which allows you to show only those emails for a specific account.

    • Stop deleting! -- This allows you to immediately halt the process of deleting a series of emails. Emails are deleted one by one, and any emails that are deleted before you have a chance to click on this option will remain deleted. But those emails that haven't been deleted yet will remain. If you need this, you'll have to be quick about it!

    • Preferences -- This opens up the preferences dialog where you can configure 3DME and add/remove/edit your existing email accounts.

    • Check for mail now -- Click this option if you don't want to wait for the specified time before an email account is checked. Note that if you have a specific email selected from the accounts menu that this option will only force an immediate check on that account only. If you have "-- All accounts --" selected within the accounts menu, then all accounts will be checked for new mail.

    • Mark read -- Select this option to to mark all of the new emails as read. When new mails are marked read, the notification options are cancelled (i.e. the keyboard lights and the tray icon will stop blinking, reminders will no longer sound, etc.) Note, however, that this will not close the 3D popup.

    • Display Tip-of-the-day -- Opens up the Tip-of-the-day window.

    • Display 3D popup -- Causes the 3D object to appear as if new mail has arrived. This is handy for those time you're bored, or just want to show off 3DME to your friends so they'll buy a copy. ;)

    • Select a new skin -- 3DME comes with two skins by default. This allows you to change between them. You can also go to the about box to find new skins and install them on your machine.

    • Hide -- This causes 3DME to hide itself in the tray. Once hidden, you can right-click on the icon to bring up this context menu. Double-clicking on the icon will unhide 3DME. Note that if 3DME is hidden, this option will be called "Open" rather than "Hide".

    • About -- Opens the about box for 3DME.

    • Help -- You're looking at it.

    • Check for updates -- Causes 3DME to check for any updates. This is done by connecting directly to the developer's server and checking for any new updates. This will also show you the revision history for 3DME (list of changes from version to version) so you can see what's new before you download the update. This function not available for 3DME Lite users.

    • Exit -- This will exit the application. It will not be hidden in the tray, you will need to restart 3DME if you want to continue using it.
